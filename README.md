# Amen Log (아멘 로그)

AI-powered prayer journal for Toss Webview.

## Tech Stack
- **Framework**: React + Vite
- **AI**: OpenAI (GPT-4o-mini)
- **SDK**: @apps-in-toss/web-framework
- **Animation**: Framer Motion

## Features
- AI-generated prayer responses with Bible verses.
- Toss AdMob rewarded ads integration.
- Swipe-to-delete history management.
- Custom UI following Toss design system.

## Getting Started

### Prerequisites
Create a `.env` file based on the environment:
```env
VITE_OPENAI_API_KEY=your_key_here
VITE_OPENAI_SYSTEM_PROMPT=...
```

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
